#  showDampingFluxVsPseudolength

Show Damping Flux Vs Pseudolength.


---

## Declaration
```matlab
 showDampingFluxVsPseudolength(self,options)
```
## Parameters
+ `self`  WVDiagnostics object
+ `timeIndices`  (optional) indices specifying which time indices to use (default: Inf)

## Discussion

  A plot focused on the damping flux as a function of pseudolength scale, for both exact and quadratic fluxes.
