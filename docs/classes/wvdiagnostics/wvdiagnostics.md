#  WVDiagnostics

Initializes the WVDiagnostics object, loads the wave-vortex transform and diagnostics files.


---

## Declaration
```matlab
 self = WVDiagnostics(filename,options)
```
## Parameters
+ `filename`  path to the WVModel output file
+ `options.diagnosticsFilePath`  (optional) path to the diagnostics file

## Returns
+ `self`  the constructed WVDiagnostics object
