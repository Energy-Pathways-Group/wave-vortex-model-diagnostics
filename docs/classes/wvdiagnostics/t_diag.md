#  t_diag

Get time vector from the diagnostics file


---

## Declaration
```matlab
 t = self.t_diag
```
## Returns
+ `t`  time vector from diagnostics file

## Discussion

  Reads the 't' variable from the diagnostics file. This might not be the same as the time vector in the model output file.
